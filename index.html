Bootstrap Click on MouseOver

jQuery plugin to add functionality of famous dontclick.it to bootstrap buttons

    Usage

    Connect jQuery, Bootstrap and bootstrap.clickonmouseover.js: 
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.9.1/jquery.min.js">
<script src="http://maxcdn.bootstrapcdn.com/bootstrap/3.3.1/js/bootstrap.min.js">
<link rel="stylesheet" href="http://maxcdn.bootstrapcdn.com/bootstrap/3.3.1/css/bootstrap.min.css">
<script src="javascripts/bootstrap.clickonmouseover.js">
    Initialize clickonmouseover: 
    $(function(){
  $('a').clickonmouseover();
  $('button').clickonmouseover();
  $("input[type='button']").clickonmouseover();
  $("input[type='submit']").clickonmouseover();
});
    Options: 
      $('button').clickonmouseover({
    speed: 5000,     /// animation interval in milliseconds
    color: '#000000' /// progress bar color
  });
